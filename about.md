#### Bower
```
$ bower install https://github.com/dirkpeter/jquery-ui.multiselect.widget.git
```

### init / minimal setup
```javascript
$('#mySelectField').multiselect();
```

### setup example
min-width of 200px and an infinite number of selected items displayed
```
$('#mySelectField').multiselect({
  maxItems: -1,
  min-width: 200
});
```
